🧠 Generative AI

Definition: AI that generates new content such as text, images, and code.

Key Idea: Moves from analysis → creation.

⚙️ How Generative AI Works

Training → Input Prompt → Processing → Output Generation

📦 Components of Gen AI

  • Model (LLM)
  • Training Data
  • Prompt
  • Output

🚀 Applications

Content creation, coding, design, education, automation.

⚖️ Pros & Cons

Pros: Fast, creative, scalable

Cons: Can be inaccurate, biased

🤖 Agentic AI

AI that can plan, decide, and execute tasks autonomously.

🔁 Agent Workflow

Goal → Plan → Execute → Observe → Improve

🧩 Agent Components

  • LLM (Brain)
  • Memory
  • Tools/APIs
  • Planner

📘 Notion

All-in-one workspace for notes, tasks, and databases.

✨ Notion Features

Notes, databases, task management, collaboration.
